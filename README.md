SI señor
=========

SI señor is a simple SI prefix converter.<br>
Assume that you always have basic SI unit for conversion (seconds for time, meters for length and so on).

#### Sample usage
`1.milli`<br>
`1.kilo.to_mega`<br>
`1.to_centi`<br>
